SKILLS

Over the past few years, I have developed various technical skills. In programming, I have an intermediate level in C. I’ve used it to create programs involving pointers, arrays, functions, and file handling. I am also strengthening my Java skills , particularly in object-oriented programming, thread management, and data structures.

I have solid experience with embedded systems like the Arduino Uno and the ESP8266. Through school projects, I have worked with various sensors, controlled motors (servo and stepper), displayed data on LCD screens, and transmitted this data to a mobile app.

In web and mobile development, I have used core JavaScript , the Express.js framework to build web APIs, and Twig as a PHP template engine for my web pages. I’ve also started learning TypeScript , combined with React , to create a web interface displaying real-time data.

Regarding networking, I have learned to configure services like Apache, DHCP, and DNS servers. I’ve also practiced different SSH connections and set up a VPN using OpenVPN.

Finally, I have programmed industrial PLCs from Siemens and Schneider, for example, to regulate the temperature of a box or to control a motor’s speed using PID control.

I am also comfortable using Microsoft Office 365 tools , especially Word and PowerPoint, to write reports and design presentations.
